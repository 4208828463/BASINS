Setting up BASINS Observed Water Quality Data File

The observed Water Quality Data can be provided in a DBF file for BASINS or HSPEXP+ to read. This format is useful when the data is at non-continuous interval. The DBF file can be edited or created in LibreOffice or OpenOffice (Excel has stopped saving to this format after MS Office 2007). Thie file format has only five columns, and they should be named exactly as follows.

ID, DATE, TIME, PARM, VALUE

  1. ID: This column should have the location of the data collected
  2. DATE: This column should have date in text format (mm/dd/yyyy).
  3. TIME: This column should have time in text format(HH:MM:SS).
  4. PARM: This column should have the name of the constituent.
  5. VALUE: This column should have the value recorded at the respective date and time for the respctive location and constituent.
