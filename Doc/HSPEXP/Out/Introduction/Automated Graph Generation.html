Automated Graph Generation
  1. HSPEXP+ produces "Regan Plots" - a plot of simulated phytoplankton, and benthic algae, along with multiple nutrients at all the reaches.
  2. HSPEXP+ can generate additional graphs for water quality, snow depth, frequency duration, scatter plots, etc. The specifications for these graphs must be provided in one or more comma-separated file (*.csv format).
  3. The time series for these graphs can be obtained from a WDM file, HBN file, or a DBF (BASINS Observed WQ file format) file.
  4. HSPEXP+ also outputs the dataset used for generating the graphs in tab delimited text files. Users can utilize this data for additional analysis outside of BASINS or HSPEXP+.
