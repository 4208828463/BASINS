Setting up BASINS Observed Water Quality Data File

The observed Water Quality Data can be provided in a DBF file for BASINS or HSPEXP+ to read. This format is useful when the data is at a non-continuous interval. The DBF file can be edited or created in LibreOffice or OpenOffice (Excel 2007 and newer versions do not allow saving to this format). This file format has only five columns, and they should be named exactly as follows. Any additional columns are not read by BASINS or HSPEXP+.

ID, DATE, TIME, PARM, VALUE

  1. ID: This column should identify the location of the collected data
  2. DATE: This column should identify the date in text format (mm/dd/yyyy).
  3. TIME: This column should identify the time in text format(HHMM).
  4. PARM: This column should identify the name of the constituent.
  5. VALUE: This column should identify the value recorded at the respective date and time for the respective location and constituent.
