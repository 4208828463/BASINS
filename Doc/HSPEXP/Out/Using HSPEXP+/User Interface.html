User Interface

 Up to: Using HSPEXP+

HSPEXP+ has a simple user interface. Basic explanations are provided in the User Interface itself. The user needs to browse to the UCI file for the project. The remaining options can be selected or deselected using check boxes. Any grayed-out options are unavailable, as they are under development.
