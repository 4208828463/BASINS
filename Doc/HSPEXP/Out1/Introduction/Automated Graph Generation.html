Automated Graph Generation

 Up to: Introduction, Contents

  1. HSPEXP+ can generate additional graphs for water quality, snow depth, frequency duration, scatter plots, etc. The specifications for these graphs must be provided in a comma-separated file (*.csv format) with filename "AutomatedGraphs.csv."
  2. The time series for these graphs can be obtained from a WDM file, HBN file, or a DBF (BASINS Observed WQ file format) file.
  3. HSPEXP+ also outputs the dataset used for generating the graphs in tab delimited text files. Users can utilize this data for additional analysis outside of BASINS or HSPEXP+.
